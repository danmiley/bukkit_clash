Common Issues

What's homeconvert.py?

It is a Python script (you need to install Python and run the script through command prompt or command line) to convert homes from other plugins (namely MyHomes and Warpz0r at the moment).

CommandBook doesn't seem to work.

Two common mistakes include forgetting to install WorldEdit (CommandBook needs it), or making a syntax error in one of CommandBook's configuration files (this is very fatal unfortunately, and can be caused by a single character typed in the wrong place). Try asking in IRC to see whether anyone can help you.

Where do I report bugs or suggest ideas?

Make sure you are using the latest version of CommandBook, try asking in IRC first (see "Need help?" above) if it's a problem, and if you think you've found a bug or have a great suggestion to provide, please report it to our issue tracker.

Still need help? See the ways that you can get assistance.
